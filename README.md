# About
Dormy is a home rental services app, created by Team 01 of CSC 648 - Software Engineering course.

# Technologies Used
- Node
- Express
- PostgreSQL
- PM2
- Nunjucks
- Bootstrap Material Design
- Passport.js
- AWS EC2
- AWS RDBMS
